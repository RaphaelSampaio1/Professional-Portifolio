### **GUIA: COMO ADICIONAR NOVOS PROJETOS AO PORTFÓLIO**

Para adicionar um novo projeto sem quebrar o estilo ou a legibilidade, siga estes passos:

#### **PASSO 1: Adicionar à página principal (index.html) - Featured Projects**
Localize a `<div class="projects-grid">` na seção Featured Projects e adicione um novo card:

```html
<div class="project-card project-NOVO_IDENTIFICADOR">
  <div class="project-overlay">
    <h3>NOME DO PROJETO</h3>
    <p>DESCRIÇÃO CURTA (O que faz e qual o resultado/valor gerado).</p>
    <div class="project-tags">
      <span>TECNOLOGIA 1</span>
      <span>TECNOLOGIA 2</span>
      <span>CATEGORIA</span>
    </div>
    <a href="LINK_DO_PROJETO" target="_blank" class="project-link">View Project</a>
  </div>
</div>
```

#### **PASSO 2: Adicionar à página de projetos (projects.html)**
Localize a `<div class="projects-grid">` e cole o código similar ao acima, mas com link interno se necessário.

#### **PASSO 3: No arquivo `styles.css`**
Adicione a referência da imagem na seção de projetos:

```css
.project-NOVO_IDENTIFICADOR {
  background-image: url("img/NOME_DA_IMAGEM.png");
}
```

*   **IDENTIFICADOR:** Crie um nome único sem espaços (ex: `project-bot-n8n`).
*   **LINK_DO_PROJETO:** URL do LinkedIn, GitHub ou YouTube.
*   **NOME_DA_IMAGEM:** O nome exato do arquivo na pasta `img/`.

---

### **REGRAS DE OURO (Dicas do Treinador):**
1.  **Nomes de Arquivos:** Nunca use espaços ou caracteres especiais nos nomes das imagens (ex: use `bot_whatsapp.png` em vez de `bot whatsapp novo.png`).
2.  **Legibilidade:** O estilo já está configurado com um "Overlay" (camada escura). Isso garante que, mesmo que sua imagem seja clara, o texto branco sempre será legível.
3.  **Proporção:** O card é retangular horizontal (`260px` de altura). Se a imagem for muito vertical, o CSS vai centralizá-la automaticamente para não distorcer.
4.  **Cache:** Se você trocar uma imagem e ela não mudar no site, aperte `Ctrl + F5` no navegador para forçar a atualização.
5.  **Featured Projects:** Para projetos em destaque na homepage, limite a 3-4 cards para manter o foco.
